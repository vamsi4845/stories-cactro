# Stories Cactro

An Instagram-inspired stories and posts application built with React, TypeScript, and Vite.

## Features

- **Stories View**: Instagram-like story viewing with auto-advance, progress bars, and navigation
- **Stories List**: Horizontal scrollable stories feed with user avatars
- **Posts Feed**: Social media posts with like, comment, and share interactions
- **Modern UI**: Instagram-inspired gradient background and clean design
- **Responsive**: Mobile-first design with smooth scrolling

## Tech Stack

- **React 19** - UI library
- **TypeScript** - Type safety
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Styling
- **Lucide React** - Icons

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Project Structure

```
src/
├── components/     # React components
├── pages/         # Page components
├── lib/           # Constants and data
├── types/         # TypeScript type definitions
└── index.css      # Global styles
```

## Features in Detail

- **Story Navigation**: Click left/right sides or use arrow keys to navigate between stories
- **Auto-Advance**: Stories automatically advance after 5 seconds
- **Pause on Hover**: Hover to pause story playback
- **Progress Indicators**: Visual progress bars showing story completion
- **Interactive Posts**: Like posts, view comments, and interact with content
